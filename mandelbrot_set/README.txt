##Purpuse:##

this project is about implementing the algorithme
of mandelbrot set, 
and plotting a madelbrot set  obtained

##Usage:##

-execute cmd :make (the mandel exec will be generated)
-execute cmd :./mandel [-option] {width}{height}
(you obtain mandel.ppm)
-execute cmd : display mandel.ppm(enjoy)

options are :

-h display help
-n black and white mandel
